# 📐 Toán Lớp 7 - Chân Trời Sáng Tạo

## 1. Số Hữu Tỉ

### Định Nghĩa
- Số hữu tỉ: Số có dạng a/b (a, b ∈ ℤ, b ≠ 0)
- Ký hiệu: ℚ
- Tính chất: Có thể biểu diễn dưới dạng số thập phân hữu hạn hoặc vô hạn tuần hoàn

### So Sánh Số Hữu Tỉ
- Cách 1: Quy đồng mẫu số
- Cách 2: So sánh với 0
- Cách 3: Quy đổi ra số thập phân

## 2. Phép Toán với Phân Số

### Cộng - Trừ Phân Số
- **Cùng mẫu**: a/b ± c/b = (a ± c)/b
- **Khác mẫu**: Quy đồng mẫu trước, rồi cộng/trừ tử

### Nhân - Chia Phân Số
- **Nhân**: (a/b) × (c/d) = (a×c)/(b×d)
- **Chia**: (a/b) ÷ (c/d) = (a/b) × (d/c) = (a×d)/(b×c)

### Tính Chất
- Giao hoán: a + b = b + a; a × b = b × a
- Kết hợp: (a + b) + c = a + (b + c)
- Phân phối: a×(b + c) = a×b + a×c

## 3. Luỹ Thừa

### Định Nghĩa
- a^n = a × a × ... × a (n lần)
- a: Cơ số; n: Số mũ

### Công Thức
- a^m × a^n = a^(m+n)
- a^m ÷ a^n = a^(m-n)
- (a^m)^n = a^(m×n)
- (a×b)^n = a^n × b^n
- (a/b)^n = a^n / b^n
- a^0 = 1 (a ≠ 0)
- a^(-n) = 1/a^n

### Luỹ Thừa Bậc 2, 3
- a² = a × a (diện tích hình vuông)
- a³ = a × a × a (thể tích lập phương)

## 4. Hình Học Phẳng

### Tam Giác
- **Chu vi**: P = a + b + c
- **Diện tích**: S = (1/2) × h × a (h: chiều cao, a: đáy)
- **Định lý Pythagoras**: a² + b² = c² (tam giác vuông)

### Hình Chữ Nhật
- **Chu vi**: P = 2(a + b)
- **Diện tích**: S = a × b

### Hình Vuông
- **Chu vi**: P = 4a
- **Diện tích**: S = a²

### Hình Tròn
- **Chu vi**: C = 2πr = πd (r: bán kính)
- **Diện tích**: S = πr²

## 5. Góc & Đường Thẳng

### Các Loại Góc
- **Góc nhọn**: 0° < α < 90°
- **Góc vuông**: α = 90°
- **Góc tù**: 90° < α < 180°
- **Góc bẹt**: α = 180°

### Đường Thẳng Song Song
- Kí hiệu: a ∥ b
- Nếu a ∥ b và b ∥ c thì a ∥ c

### Hai Đường Thẳng Cắt Nhau
- Tạo 4 góc
- Các góc đối đỉnh bằng nhau

## 6. Thống Kê & Xác Suất

### Trung Bình Cộng
- X̄ = (x₁ + x₂ + ... + xₙ) / n

### Tần Số & Tần Suất
- Tần số: Số lần xuất hiện
- Tần suất = Tần số / Tổng số

### Mốt (Mode)
- Giá trị có tần số cao nhất

## 📌 Lưu Ý Quan Trọng
1. **Thứ tự phép tính**: Lũy thừa → Nhân/Chia → Cộng/Trừ
2. **Quy tắc dấu**: (+) + (+) = (+); (-) + (-) = (-); (+) + (-) = ?
3. **Giá trị tuyệt đối**: |a| = a nếu a ≥ 0; |a| = -a nếu a < 0
4. **Số đối**: a + (-a) = 0
5. **Số nghịch đảo**: a × (1/a) = 1
