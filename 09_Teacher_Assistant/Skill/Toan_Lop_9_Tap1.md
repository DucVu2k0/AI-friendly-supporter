# 📐 Toán Lớp 9 - Tập 1: Căn Bậc 2 & Phương Trình Bậc 2

## 1. Căn Bậc 2

### Định Nghĩa
- **Căn bậc 2 của a**: √a = x nếu x² = a (a ≥ 0)
- **Căn bậc 2 số học**: Số không âm x sao cho x² = a
- **Ký hiệu**: √a (a ≥ 0)

### Tính Chất
- √a² = |a| = a (nếu a ≥ 0)
- √(a×b) = √a × √b (a, b ≥ 0)
- √(a/b) = √a / √b (a ≥ 0, b > 0)

### Khử Mẫu, Trục Căn Thức
- a/√b = (a√b)/b
- Nhân tử và mẫu đều cho √b để khử căn ở mẫu

### Căn Bậc 3
- **Định nghĩa**: ³√a = x nếu x³ = a
- **Khác với căn bậc 2**: Có thể lấy căn bậc 3 của số âm
- ³√(a×b) = ³√a × ³√b

## 2. Phương Trình Bậc 2

### Dạng Chuẩn
- **Phương trình bậc 2**: ax² + bx + c = 0 (a ≠ 0)
- **Phương trình bậc 2 khuyết**: Thiếu b hoặc c

### Công Thức Nghiệm
**Δ = b² - 4ac** (Biệt thức)

Nếu Δ > 0: 2 nghiệm phân biệt
- **x₁ = (-b + √Δ) / (2a)**
- **x₂ = (-b - √Δ) / (2a)**

Nếu Δ = 0: 1 nghiệm kép
- **x = -b / (2a)**

Nếu Δ < 0: Vô nghiệm thực

### Công Thức Rút Gọn (Δ' = (b/2)² - ac)
- **x₁,₂ = (-b/2 ± √Δ') / a** (khi b chẵn)

### Định Lý Viet
- **Tổng nghiệm**: x₁ + x₂ = -b/a
- **Tích nghiệm**: x₁ × x₂ = c/a
- **Ứng dụng**: Tìm 2 số khi biết tổng và tích

### Phương Trình Khuyết
1. **ax² + c = 0**: x = ±√(-c/a) (nếu -c/a > 0)
2. **ax² + bx = 0**: x(ax + b) = 0 ⟹ x = 0 hoặc x = -b/a

## 4. Hàm Số Bậc 2

### Hàm y = ax² (a ≠ 0)
- **Đồ thị**: Parabol qua gốc tọa độ O
- **Tính chất**:
  - a > 0: Parabol mở lên, đỉnh là điểm thấp nhất
  - a < 0: Parabol mở xuống, đỉnh là điểm cao nhất
- **Trục đối xứng**: Trục Oy

### Hàm y = ax² + bx + c
- **Đỉnh**: x = -b/(2a); y = -Δ/(4a)
- **Trục đối xứng**: x = -b/(2a)
- **Giao với Oy**: (0, c)
- **Giao với Ox**: Cho y = 0, giải ax² + bx + c = 0

## 5. Hình Học - Đường Tròn (Phần 1)

### Định Nghĩa & Tính Chất
- **Đường tròn**: Tập hợp tất cả điểm cách O khoảng r
- **Bán kính**: r
- **Đường kính**: d = 2r
- **Chu vi**: C = 2πr = πd
- **Diện tích**: S = πr²

### Vị Trí Tương Đối: Điểm & Đường Tròn
- Điểm ngoài đường tròn: d > r
- Điểm trên đường tròn: d = r
- Điểm trong đường tròn: d < r

### Vị Trí Tương Đối: Đường Thẳng & Đường Tròn
- **Cách xa**: d > r (không cắt)
- **Tiếp xúc**: d = r (tiếp tuyến)
- **Cắt**: d < r (cắt tại 2 điểm)
